How to use:
Adding a new item to the to-do list.
1. Create a new category typing a category into the first textbox.
2. Select a category and type something in the second textbox.
3. Click add to add to to-do list.
NOTE: If a category isn't select, item will be placed in NULL.
Editing an item on the to-do list.
1. Select the item that you want to edit.
2. Select the category you'd like to place the item in.
3. Type the name that you like to set the modify item to.
4. Click EDIT
Delete an item
1. Select an item you'd like to delete
2. Click DEL
Delete a category
1. Select a category.
2. Click Delete.
NOTE: Any items in the category will be moved to null.
NOTE: NULL can't be delete.
